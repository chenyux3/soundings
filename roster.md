# Soundings — Roster

The council's advisors. Each advisor is characterized by the **method of inquiry** it imposes on a question — not by a real person, a celebrity, or a fixed body of work. Frameworks are pulled dynamically per session (see `SKILL.md` Phase C), so an advisor is a *lens*, not a script.

The roster below is a **seed — one person's setup.** Treat it as an example, not a prescription. The advisors here reflect a particular life stage (independent builder, founder questions, a young child). Replace, remove, or add advisors so the council fits *your* decisions. See `ROSTER_GUIDE.md` for how.

Extensibility: add an advisor when a class of question repeatedly falls outside the current set. Adding an advisor = appending a section below; no `SKILL.md` change needed.
